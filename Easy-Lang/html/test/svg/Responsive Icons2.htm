Responsive Icons
About the Project

'Responsive Icons' is a project that explores scalable iconography. The concept interrogates the varying complexity of symbols used on devices, and aims to find the perfect balance of simplicity in relation to screen size. The project is currently in its early stages and I will be adding additional icons to the site shortly. Best viewed on desktop (scale your browser width to see the adaptions).
